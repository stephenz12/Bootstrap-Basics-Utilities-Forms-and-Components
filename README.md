Features
 1. Registration Form

Built using Bootstrap form components

Includes fields for:

First Name

Last Name

Email (required, with validation)

Password (required, with validation)

Terms & Conditions checkbox

Uses Bootstrap Grid (row, col-md-6) to place First and Last Name side-by-side

Submit button styled with btn-success

 2. User Data Table

Responsive table wrapped in table-responsive

Styled using:

table-striped

table-hover

Includes hard-coded sample user data to simulate form submissions

 3. Responsive Images

Full-width banner image using:

img-fluid

w-100

Circular profile-style image using:

rounded-circle

Both images scale smoothly across screen sizes

 4. Buttons With Utility Classes

One always-visible button

Second button hidden on small screens using:

d-none

d-md-block

 5. Responsive Navigation Bar

Built with navbar, navbar-expand-lg, and navbar-dark bg-dark

Includes links for:

Home

About

Contact

Collapses into a hamburger menu on mobile screens

 Technologies Used

HTML5

Bootstrap 5 (CDN)

Optional CSS for customization

Responsiveness

All components are fully responsive using Bootstrap utilities, including grid layout, responsive images, visibility classes, and navbar collapse behavior.
